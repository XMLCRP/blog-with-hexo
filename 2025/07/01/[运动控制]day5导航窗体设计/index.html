三轴自动搬运加工机械手项目实战开发-day5.导航窗体实现

发表于2025-07-01|更新于2025-07-08|上位机
|浏览量:

1. 创建一个登录窗体

  1. 修改Font字体和大小，该属性会影响到窗体内所有的控件的默认值
  2. 修改FormBorderStyle=None，该值会影响到边框，最大化最小化和删除按钮。
  3. 设置登录窗体继承的父类为DockContent public partial class FrmNavi:DockContent

2. 添加导航按钮

  1. 添加一个button控件，给该控件添加一个image
  2. 设置image Align=TopCenter，顶部对齐
  3. 设置Text Align=BottomCenter，底部对齐
  4. 设置FlatStyle=flat，无特殊渲染。
  5. 设置FlatApperance选项，展开里面有个BorderSize=0，边框设置为0。
    枚举值 含义 效果
    Standard 使用系统默认渲染样式 Windows 主题按钮风格（带阴影、边框）
    System 委托给操作系统绘制 完全由操作系统绘制，跟 Standard 很像，但优先使用 OS 当前主题
    Flat 扁平化 没有立体边框，纯色填充，看起来像老版 Windows 的扁平按钮
    Popup 弹出式 平时扁平，鼠标悬停时出现 3D 边框
  6. 设置closebuttonvisible属性=false,取消显示关闭按钮，防止用户误操作。
  7. 设置showhint属性=DockLeft停靠到左侧，如果是希望显示在下方就选择DockButton

3. 添加第三方控件

添加第三方控件首先要引用dll，这里不赘述。
引用完dll并不会直接出现在工具箱中，需要手动添加，添加方式有两种：

  1. 点击”工具”->”选择工具箱项”->”浏览”->选择我们要添加的dll.
  2. 直接将dll拖到左侧工具箱中。

4. 在主窗体里打开导航窗体

主窗体和导航窗体是两个独立的.cs窗体项目，注意：两个窗体的字体和字号大小必须保持一致，不然会出现界面显示不正确的问题

主窗体里面需要先添加一个Dockpannel控件命名为DockMain，这个控件是WeifenLuo.WinFormsUI.Docking.dll里面的新增控件。

  1. 设置FrmMain的IsMdiContainer属性为true，如果不设置的话会报错。
  2. 设置DockMain的Dock属性为Fill，让它占据满主窗体。
  3. 主窗体Main函数中添加代码

5.总结

知识点 核心内容 考试重点/易混淆点
导航栏窗体创建 创建独立窗体 Navigation FM，设置字体（黑体 10.5）、无边框 (FormBorderStyle=None)、固定大小 (120×650) 及图标 窗体属性设置顺序与继承关系（需成为 DockContent 而非普通 Form）
导航按钮设计 通过 Button 控件实现：- 图片对齐 (ImageAlign=Top)；- 文字对齐 (TextAlign=Bottom)；- 扁平化样式 (FlatStyle=Flat)；- 去边框 (FlatAppearance.BorderSize=0) 按钮样式统一性与动态图标匹配（如集中监控 → monitor 图标）
窗体嵌入主界面 代码逻辑：- 主窗体 Load 事件调用 Show() 方法；- 传递 DockPanel 参数实现停靠；- 设置 Dock=Left 和 AutoHidePortion 比例 (110/1280≈0.086) 关键点：Dock 和 AutoHidePortion 像素值与自动隐藏比例计算
交互功能预留 通过 委托 实现按钮点击切换右侧侧面板内容（下节课讲解） 委托机制与事件绑定的实现差异
